### Installation

- `git clone https://github.com/Manojj999/RN-Quiz_App`
- `yarn install`/`npm install`

### Running

---

## Emulator:

- `yarn run ios`/`npm run ios` or `yarn run android`/`npm run android`

## Actual Device

- Install Expo Client on Android Or Ios then
- `yarn start`/`npm start`
- Scan Qr code
